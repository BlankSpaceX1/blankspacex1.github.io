Projecting Population Growth in the United States Mainland using a ARIMA Machine Learning Model

This project employs an ARIMA Forecast model to predict population growth within the United States by region. It utilizes data scraped from the United States census and CDC fertility information, employing the Python BeautifulSoup package for scraping, the Numpy package for data cleaning, the Pandas library for organizing the data into a dataframe, and the Scikit-Learn package for creating the model.
